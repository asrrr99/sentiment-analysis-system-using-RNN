# Sentiment Analysis System Using RNN

A Recurrent Neural Network (RNN) based sentiment analysis system designed to classify text data (such as reviews or tweets) into positive, negative, or neutral sentiments. Built using Python and deep learning libraries.

---

## 🚀 Project Overview

This repository features an end-to-end Natural Language Processing (NLP) framework that processes sequential text data to predict sentiment. By leveraging the sequential memory capabilities of Recurrent Neural Networks, the model learns contextual relationships between words to determine the emotional tone behind the text.

---

## 🛠️ Features

* **Text Preprocessing:** Automated pipeline for tokenization, lowercase conversion, punctuation removal, and sequence padding.
* **Deep Learning Architecture:** Utilizes recurrent layers (such as standard RNNs, LSTMs, or GRUs) optimized for sequence modeling.
* **Interactive Notebook:** The entire pipeline—from data loading to training and evaluation—is fully documented and runnable inside a single Jupyter Notebook.
* **Evaluation Metrics:** Evaluates performance using key classification metrics, including accuracy and loss tracking.
